INSTRUCTIONS FOR RUNNING THE BOOKSTORE APPLICATION

To run web app in virutal environment, enter the following in the VisualStudio Code terminal from the working directory in Windows:

virtualenv venv			(to create virtual environment)
.\env\Scripts\activate		(to activate virtual environment)
python main.py



TO EXIT:
-----------

Ctrl+C 			(to stop the Python app)

deactivate	(to deactivate the virtual environment)
